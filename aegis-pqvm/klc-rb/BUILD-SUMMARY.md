## How to Build

```bash
# Option 1: Use the build script
./build-wasm.sh

# Option 2: Use npm scripts
npm run build        # All targets
npm run build:web    # Browser only
npm run build:node   # Node.js only
npm run build:bundler # Bundler only

# Option 3: Manual wasm-pack
wasm-pack build --target web --out-dir pkg --features wasm,mlkem,mldsa,serde
```

## Build Output

After building, you'll have three directories:

1. **pkg/** - Web target (for browser)
   - `aegis_klc_rb.js`
   - `aegis_klc_rb_bg.wasm`
   - `aegis_klc_rb.d.ts`

2. **pkg-nodejs/** - Node.js target
   - Same files, optimized for Node.js

3. **pkg-bundler/** - Bundler target (webpack, vite, etc.)
   - Same files, optimized for bundlers

## Usage Example

```javascript
import init from './pkg/aegis_klc_rb.js';

async function main() {
    // Initialize WASM
    const wasm = await init();
    
    // Create key lifecycle manager
    const klm = new wasm.KeyLifecycleManager();
    
    // Create policy
    const policy = new wasm.RotationPolicy(
        "my-policy",
        "MLDSA87",
        1000000,
        7776000,
        true,
        false,
        0.8
    );
    
    // Register and use
    klm.register_policy_js(policy);
    const keyId = klm.generate_key_js("my-policy");
    console.log("Key ID:", keyId);
}
```
